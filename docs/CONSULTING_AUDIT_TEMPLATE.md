# Consulting Audit Template

## Purpose

This template supports an AI workflow audit for NGO, education, SME, and personal brand clients.

The goal is to identify practical AI-assisted workflow opportunities, define a safe MVP, and produce reusable portfolio evidence without using real client data in public materials.

This template is for consulting and product discovery. It does not provide legal, compliance, financial, medical, or security assurance. Legal or compliance confirmation may be needed depending on the client, data type, sector, and jurisdiction.

## 8. Sensitivity Level

```text
Sensitivity level: public / internal / confidential / sensitive personal data / unknown

Reason:

Required handling:

Need legal / compliance confirmation? yes / no / unsure

Notes:
```

Guide:

| Level | Meaning | Handling |
|---|---|---|
| Public | Already published or approved for public use | Can be used with source checking |
| Internal | Business or project context not intended for public release | Summarize carefully; avoid public reuse without approval |
| Confidential | Client strategy, pricing, contracts, private reports | Do not upload unless explicitly approved and policy allows |
| Sensitive personal data | Health, identity, minors, service users, HR, counselling, financial, legal data | Avoid AI upload; use manual or heavily anonymized workflow |
| Unknown | Data classification is unclear | Stop and ask client to confirm |

## 18. Example: Generic SME Content Workflow Audit

```text
Client type:
SME service team

Current workflow:
Founder writes social posts manually from scattered notes and screenshots.

Business / operational goal:
Create a repeatable weekly content system without exposing client data.

Repeated manual tasks:
- turning meeting notes into post ideas
- rewriting captions for different channels
- preparing simple campaign briefs

Data involved:
Public posts, internal notes, anonymized client questions.

Sensitivity level:
Internal. Do not upload raw client messages.

AI opportunities:
- source note cleanup
- content repurposing
- campaign angle drafting

No-code / low-code options:
ChatGPT for drafting, Notion for source capture, Canva for visual output, Google Sheets for content calendar.

Human review points:
Founder checks facts, tone, privacy, and final publishing decision.

MVP recommendation:
One-source-to-five-content workflow using anonymized notes.

7-day roadmap:
Build one template, test with three safe sources, review outputs, and create a weekly publishing checklist.

30-day roadmap:
Pilot weekly content workflow, track output count and approval time, refine content pillars.

Risks:
Generic output, unsupported claims, accidental client identification.

Suggested deliverables:
Workflow audit, content pillar map, repurposing template, sample outputs, privacy checklist.

Portfolio evidence generated:
Anonymized case note showing AI-assisted content operations design.
```
